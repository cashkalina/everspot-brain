# Order Module — Models

Model documentation for the Order module. Orders are the primary sales contracts in Everspot, capturing what was purchased, by whom, at what price, and the payment arrangement.

| Model | Table | Description |
|-------|-------|-------------|
| [Order](./order.md) | `orders` | Sales contract between the cemetery and one or more customers |
| [OrderLine](./order-line.md) | `order_lines` | Individual line item on an order (purchasable, pricing, quantity) |
