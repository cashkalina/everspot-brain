# Memorial Module — Models

Model documentation for the Memorial module. Memorials are physical markers (headstones, plaques, monuments) associated with cemetery plots and the people they commemorate.

| Model | Table | Description |
|-------|-------|-------------|
| [Memorial](./memorial.md) | `memorials` | A physical memorial marker with dimensions, materials, dates, and vendor information |
| [MemorialPerson](./memorial-person.md) | `memorial_people` | A person (deceased individual) commemorated on a memorial, with name and partial dates |
