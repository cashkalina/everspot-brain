# Repetition Module

The Repetition module is Everspot's recurrence-scheduling subsystem. It defines the `Repetition` model (recurrence rules), the `Repeatable` trait (attached to any model that needs scheduling), the `RepetitionType` enum (simple/complex), and supporting `CarbonPeriod` / `RepeatCollection` utilities.
