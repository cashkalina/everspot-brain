# Mapping Models

Models in the Mapping module manage interactive cemetery maps and their spatial location data.

| Model | Table | Connection | Description |
|-------|-------|------------|-------------|
| [Map](./map.md) | `maps` | tenant | A cemetery map with base layer, config, and media attachment |
| [MapLocation](./map-location.md) | `map_locations` | tenant | A spatial element (group, property, curve) plotted on a map |
